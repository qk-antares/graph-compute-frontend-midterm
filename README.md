## 图计算前端-中期检查版

git地址：[qk-antares/graph-compute-frontend-midterm (github.com)](https://github.com/qk-antares/graph-compute-frontend-midterm)
后端地址：[qk-antares/graph-compute-backend-flask (github.com)](https://github.com/qk-antares/graph-compute-backend-flask)


安装依赖：

```bash
npm install
```

或

```bash
yarn
```

运行项目：

```bash
npm start
```
